# 1ページ目 (IPPAGE) — design-stationery-notebook

> **"Worth writing on page one."**

A design study of a simple, student-targeted stationery shop, executed under the **notebook (ruled + grid paper)** theme — selling **writing instruments exclusively**.

## Overview

| | |
|---|---|
| **Brand** | 1ページ目 (IPPAGE) |
| **Theme** | Notebook (Ruled + Grid) |
| **Catalog** | Writing instruments only |
| **Live Site (planned)** | `design.stationery-notebook.riumu.net` |

## Design Concept

- **Color**: Cream paper `#FBF8F0` × pale blue rule `#9EC5E0` × pink margin `#E89AA4`
- **Typography**: Klee One (educational handwriting face)
- **Aesthetic**: The whole page IS a notebook
- **UX**: 6 writing tools, 4-row "compare by use" guide, handwritten owner memo

## Tech

- Pure HTML + CSS Custom Properties + Vanilla JS

# 1ページ目 — design-stationery-notebook（日本語）
